# Correlação Binance

Sistema de monitoramento de correlação e convergência multi-ativos (CORR-WATCH).

## Descrição

Este projeto é projetado para monitorar correlações entre diferentes pares de criptoativos (Binance) e outros mercados (Forex, Índices Globais), identificando anomalias e oportunidades baseadas em divergências estatísticas.

## Funcionalidades

- Coleta de dados em tempo real da Binance.
- Integração com dados de Forex e Índices.
- Detecção de anomalias e desvios de correlação.
- Sistema de monitoramento de saúde e logs integrados.

## Instalação

```bash
# Clone o repositório
git clone https://github.com/Naldo2050/correlasao-binance.git

# Instale as dependências
pip install -r requirements.txt
```

## Uso

Execute o orchestrator ou as demonstrações:
```bash
python run_correlation_demo.py
```
